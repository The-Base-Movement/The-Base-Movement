# Social Media Icons - High Fidelity Collection

This directory contains a centralized library of high-fidelity, brand-accurate social media icons developed for "The Base" movement and optimized for production use.

## Contents

### 1. Vector Assets (.svg)
- **facebook.svg**: Official brand geometry with #1877F2 fill.
- **instagram.svg**: High-fidelity recreation featuring the official radial gradient.
- **x.svg**: Updated vector logo for the X platform (formerly Twitter).
- **tiktok.svg**: Advanced multi-path SVG utilizing CSS mix-blend modes to reproduce the signature 'glitch' effect.
- **youtube.svg**: Official brand red (#FF0000) and geometry.
- **whatsapp.svg**: Optimized vector for communication touchpoints.
- **linkedin.svg**: Official brand blue (#0A66C2) and vector geometry.

### 2. React Component Library
- **SocialIcons.tsx**: A ready-to-use TypeScript/React library containing all icons as modular components with customizable `size` and `color` props.

## Technical Specifications
- **Format**: SVG (Scalable Vector Graphics)
- **ViewBox**: 24x24 (Standardized)
- **Optimization**: Clean paths, minimal metadata, and production-ready code.

## Usage
These assets should be treated as the source of truth for social branding across the ecosystem to ensure 100% design consistency and performance.
