# Website UI kit — The Base Movement

A pixel-faithful recreation of the public website surface (`thebasemovement.com`).

Open `index.html` to see the assembled home page with a click-through demo
(register flow, mobile menu, dropdown).

Components:
- `BrandLine.jsx` — 3-stripe motif
- `Buttons.jsx` — gold / primary / outline
- `Navbar.jsx` — sticky top bar with logo, links, login/register
- `Hero.jsx` — full-bleed hero with cursor-spotlight image
- `PlatformSplit.jsx` — Base Ghana / Base Diaspora cards
- `PillarsSection.jsx` — Our Foundation 3-column pillars on dark
- `StatsSection.jsx` — animated counters with BrandLine
- `CTASection.jsx` — dark slab final CTA
- `Footer.jsx` — brand, links, newsletter, gradient bar
